GuildCreate

Represents the creation of a guild. This event is sent in one of three scenarios:

  1. When a user is initially connecting to lazily load and backfill information for all unavailable guilds sent in the IReady event.
  2. When a guild becomes available again to the client.
  3. When the current user joins a guild.
